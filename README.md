# Ruboty::PageSpeedInsights

[PageSpeed Insights](https://developers.google.com/speed/pagespeed/insights) handler for Ruboty.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-page_speed_insights'
```

## Usage

```
@ruboty psi <URL>
```
